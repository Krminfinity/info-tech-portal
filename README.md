# 講義ポータル

このプロジェクトは、学生が「情報技術II」および「情報技術III」の講義資料やテストにアクセスするためのシンプルなWebポータルです。

## ページURL

ポータルはこちらからアクセスできます：[https://krminfinity.github.io/info-tech-portal/]

## 🎓 教師向け管理システム

### 📝 管理画面の使い方

#### 1. 管理画面へのアクセス
1. ブラウザで `/admin/index.html` を開く
2. パスワード入力: `admin123`
3. 「ログイン」ボタンをクリック

#### 2. 講義の管理
**新しい講義を追加：**
1. 「講義」タブをクリック
2. 「新しい講義を追加」ボタンをクリック
3. 必要事項を入力：
   - **講義名**: 例）情報技術II
   - **説明**: 例）プログラミング基礎とWebデザイン
   - **アイコン**: Bootstrap アイコン名（例：laptop）
   - **公開状態**: チェックボックスで公開/非公開を選択
4. 「保存」ボタンをクリック

**講義の編集・削除：**
- 編集：講義リストの「編集」ボタンをクリック
- 削除：講義リストの「削除」ボタンをクリック

#### 3. 講義資料の管理
**資料を追加：**
1. 「資料」タブをクリック
2. 「新しい資料を追加」ボタンをクリック
3. 必要事項を入力：
   - **タイトル**: 例）第1回 HTMLの基礎
   - **URL**: リンク先URL
   - **タイプ**: 講義/演習/試験/参考資料から選択
   - **講義**: 関連する講義を選択
   - **表示順序**: 数値（小さい順に表示）
4. 「保存」ボタンをクリック

#### 4. 外部リンクの管理
**リンクを追加：**
1. 「リンク」タブをクリック
2. 「新しいリンクを追加」ボタンをクリック
3. 必要事項を入力：
   - **タイトル**: 例）MDN Web Docs
   - **URL**: リンク先URL
   - **説明**: 簡単な説明
   - **新しいタブで開く**: チェックボックス
4. 「保存」ボタンをクリック

#### 5. データのエクスポート/インポート
**エクスポート（バックアップ作成）：**
1. 「設定」タブをクリック
2. 「JSONエクスポート」ボタンをクリック
3. ダウンロードされたJSONファイルを保存

**インポート（データ復元）：**
1. 「設定」タブをクリック
2. 「ファイルを選択」でJSONファイルを選択
3. 「JSONインポート」ボタンをクリック

#### 6. サイトへの反映
**方法1: 自動反映（推奨）**
- 管理画面での変更は即座にサイトに反映されます
- LocalStorageに保存されるため、同じブラウザで表示確認可能

**方法2: GitHubへの公開**
1. JSONエクスポートでデータをダウンロード
2. `js/portal-data.js` ファイルにデータを反映
3. GitHubにコミット・プッシュ
4. GitHub Pagesが自動更新

### 🚀 具体的な運用例

#### 新学期の講義開始時
1. 管理画面で新しい講義を追加
2. 基本的な資料（シラバス等）をアップロード
3. JSONエクスポートしてGitHubに反映

#### 毎週の講義後
1. その週の講義資料を追加
2. 次週の演習課題リンクを追加
3. 必要に応じて関連する外部リンクを追加

#### 試験期間前
1. 試験範囲の資料をまとめて追加
2. 過去問や参考資料のリンクを整理
3. 学生向けの補助ツールリンクを追加

## 🚀 GitHub Pages デプロイ手順

### 初回設定
1. GitHubリポジトリの **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: main / (root)
4. **Save** をクリック

### 日常的な更新作業
1. 管理画面でコンテンツを編集
2. 「JSONエクスポート」でデータをダウンロード
3. `js/portal-data.js` を更新
4. Gitコミット・プッシュ
5. 数分後にサイトが自動更新

## 📱 PWA機能

このポータルはスマートフォンのホーム画面に追加できます：

### Androidの場合
1. Chromeでサイトを開く
2. メニュー → 「ホーム画面に追加」
3. アプリ名を確認して「追加」

### iOSの場合
1. Safariでサイトを開く
2. 共有ボタン → 「ホーム画面に追加」
3. アプリ名を確認して「追加」

## 🈺 ルビ（ふりがな）機能

難しい漢字にふりがなを表示できます：

1. ページ右下のルビトグルボタンをクリック
2. 難しい漢字に自動的にふりがなが表示
3. 再度クリックで無効化

## トラブルシューティング

### 管理画面にログインできない
- パスワード: `admin123`
- 大文字小文字を正確に入力
- ブラウザのJavaScriptが有効か確認

### 変更が反映されない
- ブラウザのキャッシュをクリア（Ctrl+F5）
- LocalStorageの内容を確認
- 異なるブラウザで確認

### データが消えた
- 定期的なJSONエクスポートでバックアップ作成
- `js/portal-data.js` にフォールバックデータあり
- GitHubの履歴から復元可能
